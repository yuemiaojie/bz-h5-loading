# 关于h5中的Loading小插件

## 如何使用？
> 下载
  `npm install bz-h5-loading`

> 初始化实例
  `config = { type: 1, text: '正在加载' }`
  `var load = new BzLoading(config)`

> 展示
  `load.show()`

> 隐藏
  `load.hide()`

> config参数
  + type: loading类型（1、2、3、4、5）,
  + text: 加载文字
